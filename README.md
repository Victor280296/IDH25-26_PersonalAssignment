This repository was created to store data and code of a personal project for the course of Introducation to Digital Humanities 2025-2026, taught by professor Margherita Fantoli.
Code and datasets can be found in distinct folders in order to rerun and augment the experiment. 
The inital data cleaning was made as a group project with Petar Soldo and Koji Okumura and can be found on this repository: https://github.com/petar-soldo/IDH2025-G5.
